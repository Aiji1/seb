Langkah-langkah Instalasi

1

Uninstall Software/Aplikasi

Jika di Device SUDAH PERNAH INSTALL maka WAJIB di unistall terlebih dahulu

2

Download Safe Exam Browser

Download Software/aplikasi sesuai device masing-masing, download dari tombol yang sudah disediakan di bawah ini atau dari AppStore

Windows macOS iPad
3

Install Aplikasi

Jalankan file installer dan ikuti petunjuk instalasi sesuai sistem operasi sampai selesai

4

Jalankan File Konfigurasi

Klik tombol di bawah untuk langsung menjalankan konfigurasi ASTS1-25.seb

File akan langsung membuka Safe Exam Browser dengan konfigurasi yang tepat

Jalankan ASTS1-25.seb Download Manual
Tips: Tombol pertama akan langsung menjalankan SEB, tombol kedua untuk download manual jika diperlukan
5

Test & Verifikasi

Lakukan testing untuk memastikan aplikasi berjalan dengan baik sebelum ujian
